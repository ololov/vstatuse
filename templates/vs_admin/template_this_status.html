  • Как грамотно добавить статус

      • Статус должен начинаться с заглавной буквы;

      • Статус должен содержать не более 140-ка символов;

      • Между словами нужно ставить только один пробел;

      • Знаки препинания (точки, запятые, двоеточия, точка с запятой) пишутся слитно с тем словом, после которого они стоят;

      • Между знаком препинания и следующим словом (если оно есть) ставится один пробел.


      Нарушение этих правил сводят к нулю шансы статуса быть опубликованным.
